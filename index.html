Heart Tracking App

The Heart Track application is an IoT-enabled web app for monitoring heart rate and blood oxygen saturation levels.

The device will periodically remind users to take heart rate and blood oxygen saturation level measurements.These measurements will then be transmitted to a web application through which users and physicians can view the data via desktop, tablet or mobile device.
